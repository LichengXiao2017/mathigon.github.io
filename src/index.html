Mathigon.io

Discover the open source JavaScript libraries that power Mathigon. Combining functions for DOM manipulation, animations, events handling, UI elements and mathematics, they make web development for modern browsers fun and easy.

Almost all functions provided by Mathigon Script are part of the M object.

Prototype Extension

Core.js extends the prototypes of JavaScript's native Array, Number and String types. These additions are perfectly "save" (non-enumerable and backwards compatible), unless when used together with another library which extends native prototypes.

Classes

core.js provides a class structure that supports inheritance and subtyping. Every class already contains .on, .off and .trigger methods for handling events.

var Vehicle = M.Class.extend({
    init: function() { ... },
    addPassenger: function() { ... }
});

var Car = Vehicle.extend({
    init: function(colour) { ... },
    startEngine: function() { ... }
});

var myCar = new Car('red');

Array Functions

Core.js and Fermat.js contain a variety of functions which can be used to dynamically create and modify arrays.

var timesTable = M.tabulate(function(x, y) { return x * y; }, 10, 10);

Parameter Checking

Unless where otherwise documented, none of the functions perform parameter checking. If a function is called with invalid argument types, the result is undefined and may lead to unexpected errors being thrown.

DOM Selection

Boost.js provides a variety of functions to select DOM elements. $, $I, $C and $T return a single element, while $$, $$C and $$T return arrays of elements. $N and $$N can be used to create new DOM elements.
